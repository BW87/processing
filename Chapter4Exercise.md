# 4장 나만의 도형 만들기

### 나만의 도형 만들기 예제 입니다.

### 저는 웃는 얼굴을 도형으로 만들어 보았습니다.

##### 소스코드 

```
size(500,500);
background(163,22,242);
noFill();
stroke(255,255,255);
strokeWeight(5);
ellipse(250,250,200,200);
ellipse(200,240,25,25);
ellipse(300,240,25,25);
ellipse(250,260,20,20);
arc(250,280,80,80,0,PI,PIE);
```
